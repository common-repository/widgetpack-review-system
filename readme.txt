The WidgetPack Review System replaces default WordPress comments with social review service to get more reviews mean more traffic and more sales.

== Description ==

WidgetPack is a Social reviews service for makes reviewing easier and help to boost your traffic and trust.
More information [https://widgetpack.com/review-system](https://widgetpack.com/review-system)
Try demo [https://widgetpack.com/demo/reviews](https://widgetpack.com/demo/reviews)

= WidgetPack for WordPress =

* Uses the WidgetPack API
* Reviews indexable by all search engines (SEO)
* Support Google Rich Snippets for Reviews
* Support schema.org/Review microformat for fast indexable
* Import/Export of existing comments as reviews
* Auto-sync of reviews with WidgetPack and WordPress database
* Review count

= WidgetPack Features =

* Social and guest login
* Pros and Cons
* Review premoderation
* Responsive design
* Review replies
* Any star colors
* Choice number and height of stars
* Sort by most recent, most helpful, highest score and lowest score
* Voting for reviews
* Email Notifications
* YouTube and Vimeo embedding, uploading images
* Real-time and powerful admin panel
* Spam protection with Google reCAPTCHA 2.0
* Moderator roles
* User ban, IP ban
* Group operations
* SSL

== Installation ==

**NOTE: It is recommended that you [backup your database](http://codex.wordpress.org/Backing_Up_Your_Database) before installing the plugin.**

1. Unpack archive to this archive to the 'wp-content/plugins/' directory inside of WordPress

  * Maintain the directory structure of the archive (all extracted files
    should exist in 'wp-content/plugins/widgetpack-review-system/'

2. From your blog administration, click on Comments to change settings
